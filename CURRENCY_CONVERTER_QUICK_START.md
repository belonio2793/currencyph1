## ğŸ”§ Troubleshooting

### "Rate not available" Error
**Problem:** Selected currency doesn't have a rate.
**Solution:** Try selecting a different currency or check if it's in the database.

### Dropdown takes too long to open
**Problem:** Slow device or network.
**Solution:** Wait for data to load, then try again.

### Search not finding my currency
**Problem:** Spelling mismatch.
**Solution:** Try:
- Just the currency code (USD)
- First 2-3 letters (eur, bit, php)
- Partial name match (bit for Bitcoin)

### Amount not calculating
**Problem:** Rates not loaded yet.
**Solution:** 
1. Wait a moment for rates to load
2. Refresh page if needed
3. Check browser console for errors
